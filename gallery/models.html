Face Collector

Using appropriated deepfaking code in conjunction with Arduino and Processing, this installation captures the faces of passersby and sorts them by yaw before displaying them in sequence. When within a reasonable range of the cameras, as determined by an ultrasonic depth sensor, the arrangement is able to capture as many as 90 frames from three angles in 1 second. This means that in just about 10 seconds the array can collect more than enough face data to create a reasonably realistic deepfake.

Made using bootstrap with hosting on github by Max Rizzuto | 2019
